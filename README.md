# UberAI Navigator

AI-powered data coding assistant for Uber's analytics teams, built with Fireworks.ai

## Features
- Natural language SQL generation
- Schema exploration
- PII & security guidance

## Setup
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Demo Questions
- "What were the top 10 cities by completed rides last week?"
- "What tables contain PII data?"
